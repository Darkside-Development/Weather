# Weather
Check any city's weather with this Bash script.
# Usage
`git clone https://github.com/Darkside-Development/Weather`<br><br>

Put your cities in `cities.txt`.<br><br>

`chmod +x weather.sh`<br><br>

`./weather.sh`
